# Data Flow Diagram (DFD)

## Level 0 (Overview)
User interacts with the system, and the system communicates with the database to store and retrieve data.

## Level 1 (Detailed Flow)

1. User logs into the system
2. System verifies authentication
3. User accesses dashboard
4. User performs actions such as:
   - Add inventory
   - Update stock
   - Manage warehouses
5. System updates database accordingly
