# System Modules

The system is divided into different functional modules:

## 1. Authentication Module
Handles user login and registration using secure JWT authentication.

## 2. Inventory Management Module
Allows users to add, update, delete, and view inventory items.

## 3. Warehouse Management Module
Manages warehouse details such as location and capacity.

## 4. Stock Transfer Module
Handles movement of items between warehouses.

## 5. Reporting Module
Generates reports for inventory and stock levels.
